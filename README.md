# ☕ Eclipse Café – Café Management Web Application

## 📌 Overview
**Eclipse Café** is a database-driven **Café Management Web Application** developed using **PHP** and **MySQL**.

The system simulates a real-world café ordering experience, allowing users to browse menu items dynamically and place orders through an interactive interface. It ensures efficient data handling through a structured relational database.

This project was developed as part of the **Developing Modern Web (DMW)** coursework for the *Diploma in Software Engineering*.

---

## 🎯 Key Features

### 🍽️ Dynamic Menu System
- Menu items retrieved directly from the database  
- Displays item name, description, price, and category  
- Easily extendable with new menu entries  

---

### 🛒 Order Management
- Users can place orders through the interface  
- Orders are stored securely in the database  
- Tracks order details for processing  

---

### 🔐 Admin Functionality
- Admin account logic implemented  
- Secure access to system-level features  
- Foundation for future admin dashboard  

---

### 📩 Contact System
- Users can send messages via contact form  
- Messages stored in database for review  

---

## 🛠️ Technology Stack

- **Backend:** PHP  
- **Database:** MySQL  
- **Frontend:** HTML, CSS  
- **Scripting:** JavaScript  

---

## 🗄️ Database Design

The system uses a structured relational database with the following tables:

- **Menu Items** → Stores item details (name, description, price, category)  
- **Orders** → Stores customer order information  
- **Admin Accounts** → Stores admin login credentials  
- **Contact Messages** → Stores user inquiries  

### Key Highlights:
- Normalized database design  
- Efficient data storage and retrieval  
- Sample data included for testing
